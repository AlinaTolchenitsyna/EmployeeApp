### База данных
Создать из скрипта (СкриптСозданиеБД.sql)

### Бэкенд (API)
1. Открыть `EmployeeAPI.sln` в Visual Studio 2022
2. Убедиться, что строка подключения в `appsettings.json` корректна
3. Запустить API

### Фронтенд
```bash
cd employee-app
npm install
ng serve
```
Приложение будет доступно на http://localhost:4200
